## GENERAL IMAGE SPECIFICATIONS

### Technical Requirements:
- **Format:** JPG (preferred for photos) or WebP (for web optimization)
- **File Size:** Under 200KB per image (optimize for web)
- **Resolution:** High quality but compressed
- **Aspect Ratios:** Portrait (3:4), Landscape (4:3), or Square (1:1)

### SEO Best Practices:
1. **Descriptive file names** - Use keywords: "sarvesh-mishra-[context].jpg"
2. **Alt text** - Always include descriptive alt text
3. **Loading attribute** - Hero images use `loading="eager"`, others use `loading="lazy"`
4. **Compression** - Optimize images before uploading (use tools like TinyPNG)

### Image Style Recommendations:
- **Professional portraits** - Serious, calm expression
- **Clean backgrounds** - Minimal distractions
- **Consistent lighting** - Natural or professional lighting
- **Color palette** - Align with website's muted, premium aesthetic
- **No excessive editing** - Keep authentic and grounded
